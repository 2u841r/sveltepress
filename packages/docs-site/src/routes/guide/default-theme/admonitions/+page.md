## Grammar

This feature is provided by [Remark Admonitions](https://github.com/elviswolcott/remark-admonitions)

```md
:::tip|info|note|warning|important|caution[Title]
Some admonition content
:::
```

## Tip

**Input** 

```md
:::tip[Tip title]
Some tip content
:::
```

**Output**

:::tip[Tip title]
Some tip content
:::
