## 语法

这个特性集成了 [remark-directive](https://github.com/remarkjs/remark-directive) 

```md
:::tip|info|note|warning|important|caution[标题]
一些高亮内容
:::
```

## 提示

```md live
:::tip[提示标题]
一些提示内容
:::
```
